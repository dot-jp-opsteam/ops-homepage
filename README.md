## 🔄 編集の基本サイクル（詳しくは GITHUB_GUIDE.md）

```
① Pull（最新を取り込む）→ ② VS Codeで編集 → ③ 保存
   → ④ Commit（変更に名前をつける）→ ⑤ Push（公開反映）
```
👉 **Push すると数十秒で公開サイトに自動反映**されます。

---

## 📌 チームのお約束（事故防止）

1. **作業の前に必ず Pull、終わったら Push**
2. **自分の担当ページ中心に編集**（別ファイルなら衝突しない）
3. **みんな共通のファイル**（`index.html`・ヘッダー/フッター・`css/`・`js/`）を触る時は、**一声かけてから**
4. **枠（デザイン・動き）は共通のまま／中身は各自**（→ 詳細は EDITING_GUIDE.md）
