## 🛠 みんなが使う道具

1. **GitHub Desktop** ＝ パソコンとネット(GitHub)をつなぐ"運び屋"
2. **VS Code** ＝ HTML/CSSを編集するエディタ（＋拡張機能「Live Preview」で見た目を確認）

## 🔄 編集の基本の流れ（これだけ覚えればOK）

**最初の1回だけ：** GitHubから自分のPCに「クローン」（コピーを持ってくる）

**毎回の作業：**

```
① Pull（最新を取り込む）→ ② VS Codeで編集 → ③ 保存
   → ④ Commit（変更に名前をつける）→ ⑤ Push（GitHubへ送る）
```

👉 **Push すると、数十秒で公開サイトに自動反映**されます。
